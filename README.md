# bilibili_blacklist

粉丝量<1万，黑名单上限为1000；粉丝量≥1万，黑名单上限为10000

执行程序，一键拉黑机器人/麦片人/水军号


## 贡献&使用指南

下载本项目到本地

运行main.py

根据菜单提示使用

可以贡献你的黑名单(通过运行main.py)
